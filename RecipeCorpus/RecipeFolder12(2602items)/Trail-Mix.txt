Title: Trail Mix
Categories: Try it
Yield: 2 Cups
MMMMM--------------------COOKING LIVE #CL8962-------------------------
1/4 c  Nuts; unsalted
1/4 c  Sunflower seeds
1/4 c  Cranberries, dried
1/4 c  Cherries, dried
1/4 c  Apricots, dried
1/4 c  Raisins
1/4 c  Mini chocolate chips
1/4 c  Mini pretzels
1/4 c  Cheerios, multi grain
1/4 c  Chex
1/4 c  Granola
In a bowl combine any 8 items. Store in an airtight container.
